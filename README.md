# 技术栈

主要技术栈为 React / react-router / typescript / styled-components
配合 antd 组件库进行页面搭建

# 功能简介

一个简单的记账 APP 的项目，拥有选择标签并记账的功能。
可以在标签页进行标签的增删改，可以在统计页面查询收支总额。
是在学习 React 过程中做的 Demo 项目
